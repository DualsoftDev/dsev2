# HW 시스템 조건 및 액션 사양 V3

## 1. 개요
하드웨어 시스템의 상태 조건(Ready/Drive), 제어 액션(Emergency/Pause) 및 Abnormal 조건 정의하는 프레임워크

## 2. 타입 정의

- `DuReadyState`: 시스템 준비 상태 조건
- `DuDriveState`: 시스템 구동 상태 조건
- `DuEmergencyAction`: 비상 정지 액션
- `DuPauseAction`: 일시 정지 액션
- `DuAbnormalCondition`: 이상 발생 조건

## 3. 핵심 클래스

### 3.1 ConditionDef (조건 정의)
**용도**: 시스템 상태 전환을 위한 하드웨어 조건 모니터링

**주요 속성**:
- `ConditionType`: 조건 타입 (Ready/Drive)
- `ValueSpec`:  조건 정의

### 3.2 ActionDef (액션 정의)
**용도**: 시스템 조건에 따라 실행되는 하드웨어 액션

**주요 속성**:
- `ActionType`: 액션 타입 (Emergency/Pause)
- `ValueSpec`:  액션 실행 값 정의(범위입력예외)

### 3.3 AbnormalDef (이상 정의)
**용도**: 시스템 조건에 따라 실행되는 하드웨어 액션

**주요 속성**:
- `AbnormalType`: 이상 타입 (아직은 None)
- `ValueSpec`??:  ExternalStart 와 같은 수식필요
